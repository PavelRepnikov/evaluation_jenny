# TTS Model Evaluation

This repository contains code to evaluate the quality of synthesized audio generated by the `Parler-TTS-mini-jenny-30H` model. The evaluation includes transcription of generated audio, calculation of key metrics (Word Error Rate, Mel Cepstral Distortion, Signal-to-Noise Ratio), and a summary of results.
